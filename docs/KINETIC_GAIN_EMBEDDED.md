# Kinetic Gain Embedded

This repo demonstrates the BigQuery query-cost control-plane shape of Kinetic Gain Embedded:

- operator-safe visibility into query scan spikes, slot pressure, attribution coverage, storage posture, and telemetry freshness
- reusable analyzer logic that can sit behind an embedded customer dashboard
- explicit owner/action packets that make cost governance productizable inside a larger platform

For embedded product work, the same primitive becomes an in-app FinOps or data-platform panel where operators need evidence-rich query-cost posture without direct access to raw admin consoles or live BigQuery credentials.
